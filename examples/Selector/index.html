Select 选择器 #

常用的操作按钮。

基础用法 #

适用广泛的基础单选 v-model 的值为当前被选中的option 的 value 属性值

未找到匹配的内容
展开示例代码
vue
<template>
  <a-select v-model="value" :options="options" placeholder="请选择一个选项">
  </a-select>
</template>

<script lang="ts" setup>
import { ref } from "vue";
import { YSelect } from "amu-ui";
const value = ref();
const options = ref([
  {
    value: 1,
    text: "选项 一",
  },
  {
    value: 2,
    text: "选项 二",
  },
  {
    value: 3,
    text: "选项 三",
  },
  {
    value: 4,
    text: "选项 四",
  },
  {
    value: 5,
    text: "选项 五",
  },
]);
</script>
