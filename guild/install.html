安装 #

环境支持 #

由于 Vue 3 不再支持 IE11，amu-ui 也不再支持 IE11 浏览器。

使用包管理器 #

建议您使用包管理器 (NPM, Yarn, pnpm) 安装 Amu-UI

# 选择一个你喜欢的包管理器

# NPM
$ npm install amu-ui --save

# Yarn
$ yarn add amu-ui

# pnpm
$ pnpm install amu-ui
